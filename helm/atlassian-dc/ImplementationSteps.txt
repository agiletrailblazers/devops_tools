1. Create storage class by using the file storageClass.yaml
  AKS:  kubectl apply -f storageClass_AKS.yaml
  EKS: kd apply -f storageClass_EKS.yaml
2. Create postgres for jira using helm.
    a. Navigate to folder helm/atlassian-dc
    b. Create pvc claim seperatly
        kubectl -f postgresjira/postgres-pvc.yaml -n atlassian apply

    c. helm install -n atlassian postgresjira postgresjira/
    d. helm upgrade -n atlassian --install postgresjira postgresjira/
